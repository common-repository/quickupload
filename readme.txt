Drag&Drop media uploader dashboard widget.


== Description ==

This plugin will embed the drag&drop media uploader right into a dashboard widget. So updating media can be done with out going to the media library add new media page in the admin dashboard.


== Installation ==

1. Install QuickUpload either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to your dashboard to see the widget.
